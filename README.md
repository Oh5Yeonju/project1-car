## 8. 컨벤션

| 커밋 컨벤션                                             |
| ------------------------------------------------------- |
| Feat : 새로운 기능 추가                              |
| Design : CSS, 사용자 UI 디자인 변경                  |
| Docs : 문서 수정                                     |
| Style : 코드포맷, 세미콜론, 개행, 코드 구조, 형태    |
| Test : 테스트                                        |
